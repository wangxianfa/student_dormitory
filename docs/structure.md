# 项目结构说明

- `config/` 配置文件
  - `configurable/`  可配置文件目录
  - `serverConfig`  服务启动配置
  - `sbServer`  mongodb连接配置

- `docs/` 文档说明
  - `structure.md`  项目结构说明
  - `install.md` 项目构建运行说明

- `models/` mongoose Schema文件

- `public/` 静态资源文件夹

- `router/` 后端路由服务文件

- `src/`  项目源文件目录

- `app.js`  服务入口文件

- `nodemon.json`  nodemon配置文件
